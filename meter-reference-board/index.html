05

Key Takeaways for Crew

Shoot-Day Quick Reference

  • Color palette: Deep blue/teal base with red accents from METER branding. Amber/yellow as warm counterpoint.
  • Lighting: Primarily practical/motivated — fluorescent tubes, window light, neon. Avoid flat studio lighting.
  • Camera: Handheld, intimate, DVX/camcorder look. Fisheye lens for specific shots. Tapeless adapter for texture.
  • Locations: The Subway venue — use the actual train car interiors, tunnels, platforms.
  • Movement: Reference Papercut — tracking subjects through spaces, following from behind, push-ins on faces.
  • Tone: Gritty but beautiful. Y2K nostalgia without irony. Authentic analog processes.
