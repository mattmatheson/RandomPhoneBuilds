Black Ops Zombies on Retroid Pocket 5

The full guide to running Call of Duty zombies on a handheld using Winlator and Plutonium — from PC setup to pocket gameplay.

Pro tip: For a better voice, select all then use iOS Speak Selection (Settings > Accessibility > Spoken Content > Speak Selection).

You've got a brand new Retroid Pocket 5 and you want to play Black Ops Zombies on it. This guide walks through everything — what each piece of software does, how to connect your RP5 to your PC, how to set up Winlator, how to get Plutonium running, and what to realistically expect. Most of this happens on your Windows PC first, then you transfer to the handheld.

Part 1

Part 2

Understanding the Software Stack

There are three pieces of software involved. Here's what each one does and why you need it:

Winlator — Windows on Android

Winlator is an Android app that creates a fake Windows environment on your device. Under the hood, it uses Wine (a Windows compatibility layer), Box64 (translates x86 PC instructions into ARM instructions your phone processor can understand), and Turnip Vulkan drivers (lets the GPU handle DirectX games). Think of it as tricking Windows games into thinking they're running on a real PC.

Current version: Winlator 11.0 (February 2026). Download it from the official GitHub at github.com/brunodev85/winlator or winlator.org.

Plutonium — The CoD Client

Plutonium is a free, community-made client for older Call of Duty games. The official CoD servers for these games are dead — Plutonium brings them back. It supports:

  • World at War (T4) — zombies included
  • Black Ops 1 (T5) — zombies included
  • Black Ops 2 (T6) — zombies included
  • Modern Warfare 3 (IW5)

Plutonium needs the actual game files to work. It's a launcher/modifier, not the game itself. You still need a copy of Black Ops from Steam or wherever.

The Chain

So the chain looks like this: RP5 (Android) runs Winlator which creates a Windows environment which runs Plutonium which launches Black Ops Zombies. That's three translation layers. It's ambitious.

Part 3

Fixing the PC Connection

Before you can transfer anything, you need the RP5 to show up on your Windows PC. If it's not appearing as a drive, work through these fixes in order:

Step 1: Switch USB Mode

This is the most common fix. When you plug in the USB cable, the RP5 defaults to charging only. You need to manually switch it:

  1. Plug the USB-C cable into the RP5 and your PC
  2. On the RP5, swipe down from the top to open the notification shade
  3. Look for a notification that says "Charging this device via USB" or similar
  4. Tap it and change to "File Transfer" or "MTP"
  5. The RP5 should now appear in Windows Explorer as a device

If you don't see the notification, go to Settings > Connected Devices > USB and change it there.

Step 2: Try a Different Cable

Many USB-C cables are charge-only — they literally don't have the data wires inside. If switching USB mode didn't help, grab a different cable. The one that came with the RP5 should work. A cable you've used to transfer files from a phone will also work.

Step 3: Try Different USB Ports

Use a port directly on the back of your PC (connected to the motherboard), not a front panel port or USB hub.

Step 4: Windows Driver Fix

If the device shows up in Device Manager with a yellow triangle:

  1. Open Device Manager on your PC (search for it in Start)
  2. Look under "Portable Devices" or "Other Devices" for the RP5
  3. Right-click > Update Driver > Browse my computer > Let me pick from a list
  4. Select "MTP USB Device"

If that doesn't work, right-click > Uninstall Device, unplug, replug, and let Windows reinstall.

Step 5: Disable USB Debugging

If you've turned on USB debugging in Developer Options, it can steal the USB connection away from file transfer mode. Turn it off temporarily, reconnect, and see if MTP works.

Alternative: microSD Card

If USB transfer is being stubborn, you can always pop a microSD card into the RP5, then use a USB card reader on your PC to load files onto the card. This bypasses all USB connection issues entirely.

Part 4

Getting the Game Files (PC Side)

This is the part that requires your Windows PC. You need two things: the actual Black Ops game files and the Plutonium launcher.

Getting Black Ops Game Files

You need a full copy of the game. The easiest way is through Steam:

  1. Open Steam on your PC
  2. Purchase/install Call of Duty: Black Ops (BO1) or Black Ops II (BO2) — whichever one you want zombies from
  3. Let it fully download and install
  4. The game files will be at: C:\Program Files (x86)\Steam\steamapps\common\Call of Duty Black Ops II\ (or similar)
  5. Game size is roughly 13 GB for BO2, slightly less for BO1
Heads Up

You need to own the game. Plutonium requires legitimate game files. The launcher itself is free, but the game files aren't included.

Getting Plutonium

  1. Go to plutonium.pw on your PC
  2. Create a free account
  3. Download the Plutonium launcher (tiny download, about 4.5 MB)
  4. Run the launcher, sign in, and point it to your game files folder
  5. It will download additional Plutonium-specific files (~1-2 GB)
  6. Test that it launches and runs Zombies on your PC first — this confirms the files are good

What to Transfer

Once everything works on your PC, you'll need to transfer to the RP5:

  • The entire game folder (~13-15 GB)
  • The Plutonium launcher files
  • The Winlator APK (if you haven't installed it directly on the RP5)

Use a microSD card for the game files — they're too big for the RP5's internal storage to handle comfortably alongside everything else.

Part 5

Setting Up Winlator on the RP5

Now we move to the handheld itself.

Installing Winlator

  1. Download the Winlator 11.0 APK — either from GitHub releases directly on the RP5's browser, or transfer it from your PC
  2. Open the APK on the RP5 to install it (you may need to enable "Install from unknown sources" in Settings > Security)
  3. Launch Winlator — first launch takes a minute while it sets up the Windows container

Creating a Container

Winlator uses "containers" — each one is basically a separate Windows installation. Set one up for Black Ops:

  1. Tap the + button to create a new container
  2. Name it something like "BlackOps"
  3. Set the resolution to 1280x720 — 1080p will tank performance
  4. Set graphics to DXVK (this translates DirectX to Vulkan)
  5. Select Turnip as the Vulkan driver (optimized for Snapdragon chips)
  6. Set the Box64 preset to "Performance" — if the game crashes later, try "Compatibility" instead
  7. Save and launch the container

Mapping Your Game Files

Inside the container, you need to tell Winlator where the game files are:

  1. Your microSD card shows up as a drive inside the Winlator Windows environment
  2. Navigate to the game files using the built-in file explorer
  3. You can also set up drive mappings in Winlator's container settings to point to specific folders on the SD card

Performance Settings on the RP5

  • Set the RP5 to High Performance mode in its system settings
  • Make sure the fan is on (smart fan or always-on) — this will get toasty
  • Close all other apps to free up RAM
Part 6

Launching Black Ops Zombies

Option A: Plutonium (Full Setup)

Inside your Winlator container:

  1. Navigate to the Plutonium launcher executable
  2. Run it — if it works, sign into your Plutonium account
  3. Select the game (Black Ops or Black Ops II)
  4. Choose Zombies mode
  5. Launch and pray
Reality Check

Nobody has publicly confirmed Plutonium working through Winlator on any Android device. The launcher requires 64-bit Windows and DirectX 11, which Winlator technically supports, but Plutonium's components might not cooperate under Wine/Box64 translation. If Plutonium's launcher fails, try Option B.

Option B: Vanilla Black Ops (Skip Plutonium)

If Plutonium doesn't work through Winlator, try running the base game directly:

  1. Navigate to the game folder inside Winlator
  2. Find the game's .exe file (like BlackOps.exe or t6zm.exe for BO2 Zombies)
  3. Run it directly
  4. For Black Ops II, you can launch the Zombies executable separately without needing the full multiplayer launcher

Winlator compatibility lists show Black Ops as a tested title with 30-60 FPS claimed on similar hardware. Without Plutonium's overhead, you might get better performance.

Part 7

Alternatives If It Doesn't Work

If the full Winlator + Plutonium setup turns out to be too unstable, here are your backup plans ranked by likelihood of success:

1. Stream from Your PC (Easiest)

Install Moonlight or Steam Link on the RP5 and stream the game from your Windows PC over your home Wi-Fi. This gives you full-quality Black Ops Zombies with zero performance issues on the handheld. Your PC does all the work — the RP5 just displays the video and sends your inputs.

  • Moonlight — works with NVIDIA GPUs (needs GeForce Experience)
  • Steam Link — works with any GPU, runs through Steam
  • Parsec — works with any GPU, slightly more setup

Downside: you need to be on the same network as your PC (or set up remote access).

2. Call of Duty Mobile

CoD Mobile has a Zombies mode and runs natively on Android. Install it from the Play Store on the RP5. It's not the same as Black Ops Zombies, but it's a genuine Call of Duty Zombies experience with no translation layers.

3. World at War Zombies (Legacy)

There was an official Call of Duty: Zombies mobile game (iOS/Android). If you can find the APK, it runs natively on Android. It's basically World at War Zombies adapted for touchscreen.

4. Lighter CoD Titles Through Winlator

If Black Ops is too heavy, older Call of Duty games run better through Winlator:

  • Call of Duty 2 — confirmed 30-45 FPS on similar hardware
  • Call of Duty 4: Modern Warfare — lighter engine, should run well
  • World at War — slightly less demanding than Black Ops

Bottom line: You need the PC for getting the game files and Plutonium. That's a one-time thing. After that, the Mac can handle file transfers if needed, but really you're doing everything on the RP5 itself.

Mac File Transfer

If you need to transfer from Mac later, install OpenMTP (free, open-source) — it replaces Google's dead Android File Transfer app. Or just use a microSD card with a USB reader, which works on both Mac and PC with zero setup.

Part 9

Quick Reference — The Full Checklist

Here's the step-by-step in order:

  1. Fix RP5 USB connection (switch to MTP / File Transfer mode)
  2. On your Windows PC: install Black Ops from Steam
  3. On your Windows PC: create Plutonium account and download launcher
  4. On your Windows PC: point Plutonium at game files, test Zombies works
  5. Transfer game files + Plutonium to RP5's microSD card
  6. On RP5: install Winlator 11.0 APK
  7. On RP5: create Winlator container (720p, DXVK, Turnip, Performance preset)
  8. On RP5: set device to High Performance mode, fan on
  9. Inside Winlator: navigate to game files, try Plutonium launcher
  10. If Plutonium fails: try running the game .exe directly
  11. If that fails: set up Moonlight/Steam Link for streaming instead
