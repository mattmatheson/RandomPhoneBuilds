Loading Methods Compared

There are several ways to load games on a PS2 without discs. Here's how they stack up for a Slim.

Method Speed Works on Slim? Notes
MX4SIO Fast Yes Best option for Slim. Near-HDD speeds.
USB Very slow Yes USB 1.1 bottleneck (~0.8 MB/s). Stutters on demanding games.
SMB Network Moderate Yes ~1.7 MB/s. Needs Ethernet cable + PC/NAS running.
Internal HDD Fastest No Fat PS2 only. Gold standard but not an option for you.

Step 1 — Get FreeMCBoot

FreeMCBoot (FMCB) is a softmod that lives on a PS2 memory card. It replaces the normal PS2 boot menu with one that can launch homebrew apps like OPL. No hardware modification at all.

Option A: Buy a Pre-Made Card (Easiest)

1

Buy an FMCB memory card

Search eBay or Amazon for "FreeMCBoot PS2 memory card." They're about $5-10 and ship with FMCB pre-installed on a legit Sony 8MB card.

2

Plug it in and boot

Insert into memory card slot 1. Turn on your PS2. Instead of the normal tower animation, you'll see the FMCB menu with options like uLaunchELF and Free MCBoot Configurator. That's it — you're modded.

Option B: DIY with FreeDVDBoot (Free)

1

Download FreeDVDBoot

Grab the ISO from the FreeDVDBoot GitHub repo (github.com/CTurt/FreeDVDBoot). Pick the version matching your PS2's DVD player version.

2

Burn it to a DVD-R

Use ImgBurn, burn at 2-4x speed. Must be DVD-R, not DVD+R or RW.

3

Set PS2 language to English

In System Configuration, set language to English. The exploit requires this.

4

Boot the disc

Start the PS2 without a disc, then insert the FreeDVDBoot DVD. It boots into uLaunchELF.

5

Install FMCB

Put the FMCB installer on a USB stick beforehand. In uLaunchELF, navigate to mass:/ (USB), run the installer, and write FMCB to your memory card. Reboot — you're done.

Step 2 — Install OPL

Open PS2 Loader (OPL) is the homebrew app that actually loads your games. It supports USB, MX4SIO, SMB, and HDD.

1

Download OPL

Get the latest OPL 1.2.0 Beta from github.com/ps2homebrew/Open-PS2-Loader/releases — download the OPNPS2LD.7z file and extract it.

2

Copy to USB

Put OPNPS2LD.ELF on a USB flash drive.

3

Transfer to memory card

Boot FMCB, open uLaunchELF (File Browser). Navigate to mass:/ (USB), copy the ELF file. Navigate to mc0:/ (memory card slot 1), paste it there.

4

Add OPL to FMCB menu

Open Free MCBoot Configurator from the FMCB menu. Add a new boot item pointing to mc0:/OPNPS2LD.ELF. Save and reboot. OPL now shows up in your boot menu.

Use the latest beta. OPL 1.2.0 Beta has years of fixes over the old v1.1.0 stable release, including exFAT support and better game compatibility. The community universally recommends the beta.

Step 3 — Set Up MX4SIO

MX4SIO is a tiny adapter that plugs a MicroSD card into your PS2's memory card slot. It's the best game loading option for Slim PS2s — near-HDD speeds, no USB 1.1 bottleneck.

1

Format your MicroSD

Format as exFAT with default allocation size. On Mac, use Disk Utility. On Windows, right-click the drive and format. exFAT has no 4GB file limit so large games like GT4 (~6.4GB) work without splitting.

2

Create folder structure

Create a DVD folder on the root of the MicroSD. This is where your PS2 game ISOs go.

MicroSD Root/ ├── DVD/ <-- PS2 DVD games (.iso) ├── CD/ <-- PS2 CD games (.iso, if any) ├── ART/ <-- Cover art (optional) └── CFG/ <-- Per-game configs (optional)
3

Plug it in

Insert the MX4SIO adapter (with MicroSD inside) into memory card slot 2. Your FMCB card stays in slot 1.

4

Enable MX4SIO in OPL

Launch OPL from the FMCB menu. Go to Settings and enable MX4SIO as a device. Save settings. Your games will now appear in the game list.

Step 4 — Add Games

Getting games onto your MicroSD is dead simple.

1

Get your ISO files

You need PS2 game ISOs. If you own the disc, you can rip it using ImgBurn on PC or a similar tool. Place the .iso files in the DVD/ folder on your MicroSD.

2

Name them clearly

OPL reads the filename as the game title. You can use plain names like Gran Turismo 4.iso or the full OPL format: SCUS_973.28.Gran Turismo 4.iso (the game ID prefix helps OPL load faster with large libraries).

3

Boot and play

Pop the MicroSD into the MX4SIO adapter, plug it into slot 2, boot OPL, and your games are right there. Select and launch.

Cover art: For box art in OPL, download art packs from the OPL community and place them in the ART/ folder. Files should be named by game ID (e.g., SCUS_973.28_COV.jpg).

Gran Turismo 4 — Specific Notes

GT4 is a dual-layer DVD game (~6.4 GB) that streams car and track assets heavily. It's one of the more demanding PS2 titles, which is why the loading method matters.

Why MX4SIO over USB for GT4

The PS2's USB ports are USB 1.1 (~0.8 MB/s actual). GT4 constantly streams assets during races, and USB 1.1 can't keep up — users report stuttering, black screens, and 40-second save times. MX4SIO avoids all of this with significantly higher throughput.

OPL Settings for GT4

If GT4 has issues, try these in OPL's per-game settings (highlight the game, press Triangle or Circle for options):

Mode 6 — Some users report needing this enabled for GT4 to boot properly.

OPL version — GT4 compatibility has varied across OPL beta builds. If one build gives you a black screen, try rolling back to a slightly older beta from the GitHub releases page.

Region

Make sure your ISO region matches your console. NTSC-U (Americas) is SCUS_973.28. PAL users report slightly better compatibility with OPL — the PAL game ID is SCES_517.19.

Troubleshooting

OPL doesn't show any games

Make sure MX4SIO is enabled in OPL settings. Check that your ISOs are in the DVD/ folder (not nested in subfolders). Verify the MicroSD is formatted as exFAT.

Game freezes or black screen on launch

Try a different OPL beta build. Some builds have regressions with specific games. You can also try enabling different compatibility modes in per-game settings.

FMCB menu doesn't appear on boot

Make sure the FMCB card is in slot 1. Check your PS2 model — SCPH-90001 and later are not compatible. Try re-seating the memory card.

MX4SIO not detected

Must be in slot 2 (slot 1 is for FMCB). Try a different MicroSD card — some cards have compatibility issues. Reformat to exFAT with default allocation size (not 512KB).

Slow load times

Make sure you're using MX4SIO and not USB. A faster MicroSD card (U3/A2 rated) can also help. If using an older/slower card, consider upgrading.
