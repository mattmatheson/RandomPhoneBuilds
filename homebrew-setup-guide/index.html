Setting Up Claude Code on a Fresh Mac

Homebrew, Node.js, and Claude Code. Three commands on a brand new MacBook.

Pro tip: For a better voice, select all then use iOS Speak Selection (Settings > Accessibility > Spoken Content).

Fresh MacBook, nothing installed? This guide walks you through every step: installing Homebrew (the Mac package manager), Node.js (the runtime Claude Code needs), and finally Claude Code itself. Copy-paste each command and you're done.

Step 1

Open Terminal

Press Command + Space to open Spotlight, type Terminal, and hit Enter. This is your command line -- where you'll paste everything below.

Terminal lives in Applications > Utilities > Terminal. You can also search for it in Launchpad. It's just a text-based way to talk to your Mac.

Step 2

Install Homebrew

Homebrew is the package manager for macOS. It lets you install developer tools with simple commands instead of hunting for downloads. Paste this into Terminal and hit Enter:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

It will ask for your Mac password (the one you use to log in). When you type it, nothing will appear on screen -- that's normal, it's just hidden. Type it and press Enter.

Important -- Don't skip this

When the install finishes, Homebrew will print a "Next steps" message with two commands you need to run. They look something like this:

echo >> ~/.zprofile echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile eval "$(/opt/homebrew/bin/brew shellenv)"

Copy and paste those exact lines from your Terminal output (they may differ slightly). This adds Homebrew to your PATH so your Mac can find it.

Verify it worked

brew --version

You should see something like Homebrew 4.x.x. If you get "command not found," close Terminal, reopen it, and try again.

Step 3

Install Node.js

Claude Code runs on Node.js. Install it through Homebrew with one command:

brew install node

This installs both Node.js and npm (Node Package Manager), which you'll use in the next step.

Verify it worked

node --version && npm --version

You should see two version numbers. Node should be v18 or higher (it will be if you just installed it). npm should be v9 or higher.

That's the hard part done

Homebrew and Node.js are the only prerequisites. Everything else is one command.

Step 4

Install Claude Code

Now install Claude Code globally using npm:

npm install -g @anthropic-ai/claude-code

This downloads and installs the Claude Code CLI so you can run it from anywhere in Terminal.

Permission error?

If you see an EACCES error, run it with sudo instead:

sudo npm install -g @anthropic-ai/claude-code
Step 5

Launch Claude Code

You're installed. Start it up:

claude

The first time you run it, a browser window will open asking you to log in with your Anthropic account. Sign in and approve the connection. Once authenticated, you're dropped into Claude Code in your terminal.

If you'd rather use an API key, set it as an environment variable instead:

export ANTHROPIC_API_KEY="your-key-here"

To make it persist across terminal sessions, add that line to your shell profile:

echo 'export ANTHROPIC_API_KEY="your-key-here"' >> ~/.zshrc
You're in

That's it. Claude Code is running. Ask it anything, point it at a project folder, or just explore.

Quick Reference

Handy commands once you're inside Claude Code:

claudeStart Claude Code
claude --helpSee all available options
/helpIn-session help
/clearClear conversation context
/compactCompress conversation to save context
/costCheck session token usage
Ctrl+CCancel current action
Ctrl+DExit Claude Code

Troubleshooting

"command not found: brew"

You missed the PATH step after installing Homebrew. Run the commands Homebrew printed under "Next steps," or close and reopen Terminal.

"command not found: node"

Close Terminal and reopen it, then try node --version again. If still nothing, run brew install node again.

EACCES permission errors with npm

Fix npm's default directory so you don't need sudo:

mkdir -p ~/.npm-global npm config set prefix '~/.npm-global' echo 'export PATH="$HOME/.npm-global/bin:$PATH"' >> ~/.zshrc source ~/.zshrc

Then re-run the Claude Code install without sudo.

Xcode Command Line Tools popup

macOS might prompt you to install Xcode Command Line Tools during the Homebrew install. Click "Install" and wait for it to finish -- it's needed for compiling some packages. This can take a few minutes.

Node version too old

If you already had Node installed and it's below v18, update it:

brew upgrade node
