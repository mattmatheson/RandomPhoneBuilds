Put It Together

Now the fun part. Your eXtremeRate shell is IPS-ready (no cutting needed) and the Hispeedido V5 is pre-laminated (no dust worries). This is a clean drop-in install.

Quick Reference: Order of Operations

V5 screen into eXtremeRate shell → 5 bezel screws → Rubber covers → Thread cable + hinges → Hinge cover screw → Connect ribbon to mobo → Speaker + buttons + pads → 3 mobo screws → Power switch → TEST POWER ON → Bottom shell + 6 screws → Battery + cover
1
V5 Screen into eXtremeRate Shell
Swap hinge caps if your V5 kit included new ones
Slide old caps off perpendicular to the hinge (may need force). New ones click on. Grooves ensure correct orientation.
Remove ALL protective films from the V5 screen
Both sides. The V5 is pre-laminated, so the lens is already bonded — no dust can get in. Clean with alcohol if needed. Make sure the foam pad on the back is attached.
Place V5 screen display-down into the inner eXtremeRate screen shell
Ribbon cable should point toward the hinge sockets. Twist/loop the cable once before routing.

The cable must loop once before threading through the hinge channel.

Place outer screen shell, install 5 tri-point screws (4.7mm)
Hold both halves firmly together, flip so screen faces up, then screw. Press in the 5 rubber covers from your eXtremeRate kit.
2
Hinges & Body
Thread cable through body shell, install hinges
Slide screen assembly hinge sockets into the body. Push hinges halfway in (they'll stop). Open fully — you should hear/feel a satisfying click. Then push the rest of the way.
No click when opening/closing? Hinges are misaligned. Pull them out and redo.
Install hinge cover + its screw (JIS #00, 6.3mm)
Close the GBA SP. Position hinge cover with screw hole nearest the screen. Hold ribbon cable aside while tightening.
3
Motherboard & Components
Reconnect the V5 screen ribbon cable
Open to 90 degrees. Make sure the connector latch is UNLOCKED (flipped up). Slide the V5 ribbon cable fully in with tweezers. HOLD the cable in place with tweezers while you push the latch closed.
The cable slides out on its own. You must hold it with tweezers the entire time you're locking the latch.
Flip motherboard back over hinges, install speaker
Place the tiny speaker mesh into the recess first, then the speaker (gold contacts facing up). Transfer the LED diffuser from old shell to eXtremeRate shell if it's not pre-installed.
Place buttons and silicone pads
A/B and brightness buttons are keyed — they only fit one way. Use your OEM buttons if they're in good shape (they usually feel better). Lay all 4 silicone pads over buttons. Lower motherboard onto the posts.
Install 3 motherboard screws (JIS #00, 4.1mm)
Make sure all posts align before tightening. Place power switch cover with notched side up.
4
Test & Close Up
TEST: Insert battery and power on NOW
Do this BEFORE closing the bottom shell. Check: does the V5 screen display? Does the touch sensor brightness work? Do all buttons respond? Much easier to fix with the shell open.
Install shoulder buttons, then the eXtremeRate bottom shell
Transfer shoulder buttons from old shell (or use eXtremeRate's). Test button feel before screwing. Install 4x 6.3mm corner screws + 2x 4.6mm inner screws (tri-point).
Battery, battery cover, final screw (JIS #00)
Insert battery at a downward angle. Slide cover tabs into cutouts. Tighten screw. Apply any replacement stickers from the eXtremeRate kit.
You're done. Power on. That pre-laminated Hispeedido V5 in the clear black eXtremeRate shell is going to look incredible.

Watch Out For

Everything that can go wrong and how to avoid it.

The Ribbon Cable. Almost no slack. Never yank it. Support the screen whenever the cable is connected. When feeding through the hinge tunnel, go SLOW. During reassembly, hold it with tweezers while locking the latch — it slides out on its own.
Hinge Alignment. If hinges don't "click" when opening/closing, they're out of alignment. Remove and redo. Best method: shell closed, push halfway in, open fully (click), then push the rest.
Stripping Screws. GBA SP uses JIS screws. A regular Phillips WILL strip them. Use iFixit PH00 or JIS #00. Push down firmly while turning slowly.
The Hinge Cover Screw. Hidden near the ribbon cable. People forget this during both disassembly AND reassembly. If the hinge area feels loose, this is why.
Test Before Closing. Always power on before installing the bottom shell. If the V5 screen doesn't light up, reseat the ribbon cable. 90% of the time that's the fix.
Keep Your OEM Buttons. The eXtremeRate buttons are fine, but your original buttons and D-pad may feel better. Try both and pick what feels right.
Don't Forget the Speaker Mesh. Tiny circular mesh that sits under the speaker. Easy to lose. Without it, debris gets inside.
Your V5 Has Touch Brightness. No need to solder a brightness wire. Just touch the sensor on the screen to cycle brightness levels. The touch pad location varies by kit — check the included instructions.
